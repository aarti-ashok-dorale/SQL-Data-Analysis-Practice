# SQL for Data Analytics 

Welcome to my SQL for Data Analytics repository!

This repository documents my journey of learning and practicing SQL with a focus on data analysis. It contains structured notes, practice queries, and mini projects designed to strengthen my understanding of relational databases and analytical problem-solving.

## Objectives

* Build a strong foundation in SQL for Data Analytics.
* Practice writing efficient and optimized SQL queries.
* Understand database design, relationships, and data integrity.
* Solve real-world business problems using SQL.
* Prepare for Data Analyst interviews and technical assessments.

## Topics Covered

### SQL Fundamentals

* Introduction to SQL
* DDL (CREATE, ALTER, DROP, TRUNCATE)
* DML (INSERT, UPDATE, DELETE)
* DQL (SELECT)
* DCL (GRANT, REVOKE)

### Database Concepts

* Constraints (PRIMARY KEY, FOREIGN KEY, UNIQUE, NOT NULL, CHECK, DEFAULT)
* Operators and Conditional Statements
* SQL Functions
* Aggregate Functions

### Querying and Analysis

* WHERE, GROUP BY, HAVING, ORDER BY
* Subqueries
* Joins (INNER, LEFT, RIGHT)
* Data Filtering and Aggregation
* Analytical Query Practice

## Tools & Technologies

* MySQL
* SQL
* MySQL Workbench

## Learning Goals

* Write clean and optimized SQL queries.
* Perform data extraction, transformation, and aggregation.
* Analyze datasets using SQL.
* Develop problem-solving skills required for a Data Analyst role.

This repository is actively maintained and updated as I continue learning SQL and applying it to data analytics use cases.
